# Product

Proof-of-Intelligence Explorer is a free hosted verification, certification, API, and badge platform for 0G iNFT agents.

It answers: is this iNFT actually intelligent?

The product acts like an Etherscan verified-contract panel, SSL certificate, and VirusTotal-style report for 0G iNFT agents. Users can verify any 0G token, inspect CodeGuardian as a proof-positive seeded agent, compare FakeAgent as a negative control, create a Passport draft, and embed a public badge.

Primary 0G alignment:

- Framework/tooling: explorer, SDK, CLI, registry, adapters, badge/API, create-passport flow.
- Autonomous/iNFT: CodeGuardian proves embedded intelligence, memory, compute evidence, replay, and certificate.
